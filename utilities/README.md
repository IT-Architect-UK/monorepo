# Utilities

Small helper scripts. Usage in each header.

| Script | Purpose |
|--------|---------|
| `git-clone.ps1` | Clone a git repository with sane defaults (PowerShell; comment-based help) |
| `github-monorepo-download.sh` | Download this monorepo without git (tarball fetch + extract) |
| `make-sh-executable.sh` | Recursively `chmod +x` every `.sh` under a given directory — handy after checkout on a fresh host |
